# Contributing

Thanks for contributing to `BinaryDecimalConverter`.

This is a small practice project, so contributions should stay simple, readable, and easy to review.

The repository license is not finalized yet, so if licensing matters for a planned contribution, discuss it before opening a larger pull request.

## Guidelines

- Keep pull requests focused on one change.
- Prefer small fixes, input validation improvements, and readability improvements.
- If a change affects behavior, describe it clearly in the commit message or pull request.
- Open a discussion first if you want to do a larger refactor.
- Be respectful and constructive during review.
